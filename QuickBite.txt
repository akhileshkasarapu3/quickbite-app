Step 2: Return a real JSON API response (health.go)

a. Set Response type to json
b. create response object
c. Convert it to json
    - json.NewEncoder(w).Encode(response)
        * NewEncoder  
          - pipe to server output
          - creates encoder and hooks it to server output

        * Encoder
          - converts the Go map into JSON and writes it into the response.
d. send it back to client
